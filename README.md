## 『はじめてのフレームワークとしての FuelPHP 第2版(改訂版)』について

本書は、最新の PHP 5.3+ フレームワークである FuelPHP のバージョン 1.7.2 に関する入門書です。

フレームワークを使ったことのない PHP ユーザの方が、FuelPHP を使いこなせるようになるための入門書です。
